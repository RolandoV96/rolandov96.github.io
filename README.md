# Proyectos

## Título: Análisis de patrones de viajes y factores climáticos para Zuber

Problema: La necesidad de comprender las preferencias de los pasajeros y el impacto de factores externos como el clima en la frecuencia de viajes de una nueva empresa de viajes compartidos.

Qué hice: Analicé una base de datos completa de viajes en taxi en Chicago, realizando análisis exploratorio de datos para identificar patrones de uso y probé hipótesis sobre el impacto del clima en la duración de los viajes desde el Loop hasta el Aeropuerto Internacional O'Hare.

Resultado o aprendizaje: Logré identificar las empresas de taxis más populares, los barrios con mayor demanda de viajes, y demostré estadísticamente cómo las condiciones climáticas adversas afectan la duración de los trayectos en rutas específicas.

Stack: SQL, Python, Análisis Estadístico, Pruebas de Hipótesis.

## Título: Análisis de segmentación de clientes para programa de fidelización

Problema: La necesidad de segmentar la base de clientes de Store 1 para diseñar campañas personalizadas y optimizar el lanzamiento de un nuevo programa de fidelización.

Qué hice: Limpié y transformé datos de perfiles de clientes, estandaricé información demográfica, calculé métricas de gasto por categoría y preparé indicadores de negocio (KPIs) para la segmentación.

Resultado o aprendizaje: Logré identificar patrones de comportamiento de compra por categorías y grupos demográficos, preparando los datos para campañas de marketing dirigidas.

Stack: Python, Análisis de Datos, Limpieza de Datos, KPIs.
